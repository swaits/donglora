# DongLoRa has moved

This repository has been split into individual repos under the
[**donglora** GitHub organization](https://github.com/donglora).

| What | New location |
|------|--------------|
| Firmware | [donglora/firmware](https://github.com/donglora/firmware) |
| Rust client library | [donglora/client-rs](https://github.com/donglora/client-rs) |
| Python client library | [donglora/client-py](https://github.com/donglora/client-py) |
| Rust mux daemon | [donglora/mux-rs](https://github.com/donglora/mux-rs) |
| Python mux daemon | [donglora/mux-py](https://github.com/donglora/mux-py) |
| MeshCore bridge | [donglora/bridge](https://github.com/donglora/bridge) |
| MeshCore AI bot | [donglora/ai-bot](https://github.com/donglora/ai-bot) |
| Example scripts | [donglora/examples](https://github.com/donglora/examples) |

This repo is archived and will not receive further updates.
